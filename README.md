# Virtual Lab Mobile

Ini adalah virtual lab mobile untuk  **https://virtuallabmike.vercel.app**


Build APK dari aplikasi ini bisa diakses di **https://drive.google.com/file/d/1UwA1zRZmpqARC4dQupKHLVWUdydSWZbY/view?usp=sharing**


Virtual Lab Mobile ini mensimulasikan pendulum interaktif (pengguna bisa mengatur panjang dari benang pendulum yang digunakan) untuk mendapatkan visualisasi dan mengetahui efeknya kepada periode pendulum.
