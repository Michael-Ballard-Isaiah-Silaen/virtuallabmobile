##Deskripsi

Ini adalah virtual lab mobile untuk  **https://virtuallabmike.vercel.app**


Build APK dari aplikasi ini bisa diakses di **https://drive.google.com/file/d/1UwA1zRZmpqARC4dQupKHLVWUdydSWZbY/view?usp=sharing**


Virtual Lab Mobile ini mensimulasikan pendulum interaktif (pengguna bisa mengatur panjang dari benang pendulum yang digunakan) untuk mendapatkan visualisasi dan mengetahui efeknya kepada periode pendulum.

# Pengoperasian

## Jika anda memakai VSCODE + Expo
1. Lakukan git clone terhadap repositori ini <br>
2. Lakukan cd ke dalam folder baru <br>
3. Pastikan anda memiliki NPM, NVM, dan package Expo (jika belum, run " npm install -g expo-cli ") di dalam sistem anda <br>
4. Lakukan command " npm i " <br>
5. Lakukan command " npx expo start -c " <br>
6. Pastikan anda memiliki aplikasi Expo Go di handphone Android/iOS anda <br>
7. Jika anda menggunakan Android, buka aplikasi Expo Go dan pilih opsi Scan QR, dan scan qr yang ada di terminal komputer anda. Jika anda menggunakan iOS, bukalah aplikasi kamera dan scan QR code yang ada di terminal komputer anda. <br>

## Jika anda ingin mencoba memakai APK langsung
1. Download dari **https://drive.google.com/file/d/1UwA1zRZmpqARC4dQupKHLVWUdydSWZbY/view?usp=sharing**
